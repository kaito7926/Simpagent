## Trust Boundaries

| Boundary | Description |
|---|---|
| Browser -> Kong -> FastAPI | Untrusted forms, cookies, and bearer access cross the public edge |
| Compose networks -> private services | Port/network declarations define exposure |
| Frontend build -> browser | Embedded credentials or token state become public |

| Threat ID | Category | Severity | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-01-08 | Information Disclosure | HIGH - blocking | frontend | mitigate | Memory-only access, server `/me`, no raw diagnostics |
| T-01-09 | Elevation | HIGH - blocking | registration UI | mitigate | No role/scope controls; server D-01/D-02 authority |
| T-01-10 | Elevation | HIGH - blocking | Compose exposure | mitigate | Private DB/backend/Admin/sandbox ports and networks |
| T-01-11 | Spoofing | HIGH - blocking | browser origin | mitigate | One public Kong origin; exact Origin/CSRF completion in 01-05 |
| T-01-SC | Tampering | HIGH - blocking | npm/images | mitigate | Lockfile and approved official images only |
</threat_model>
