## Trust Boundaries

| Boundary | Description |
|---|---|
| User/model Markdown -> browser DOM | Stored untrusted content becomes rendered UI |
| Clipboard action -> browser API | Code text is copied without executing or transforming content |
| Npm registry -> frontend runtime | New Markdown and highlighting packages become browser-executed code |

## STRIDE Threat Register

| Threat ID | Category | Severity | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-02-18 | Information Disclosure | HIGH - blocking | Markdown links | mitigate | URL scheme allowlist and external rel noopener noreferrer tests |
| T-02-19 | Elevation | HIGH - blocking | raw HTML rendering | mitigate | No rehype-raw, no dangerouslySetInnerHTML, raw HTML remains inert |
| T-02-20 | Tampering | HIGH - blocking | code block rendering | mitigate | Render fenced code as inert text with copy-only behavior |
| T-02-21 | Repudiation | LOW | sanitizer behavior | mitigate | Silent safe rendering with tests for absence of warning/noisy fields |
| T-02-SC | Tampering | HIGH - blocking | npm package install | mitigate | Blocking human legitimacy gate before package.json/package-lock update because RESEARCH.md lacks Package Legitimacy Audit table |
</threat_model>
